Features

Six phases.
Zero hiding spots.

Memory, JVM flags, loaded DLLs, classpath JARs, network connections — Crystal Scanner covers every vector cheats use to stay hidden.

01 · MEM
⬡
Deep Memory Scan

Reads every committed heap region of the Minecraft JVM, matching binary signatures using a multi-threaded Aho-Corasick automaton.

02 · JVM
◈

Scans process arguments for -javaagent injections and suspicious runtime flags used by cheat loaders and injection frameworks.

How it works

Under
the hood

A precise multi-phase pipeline from process discovery to a forensic-grade report.

01
Process Discovery

Scans all running processes for Java-based Minecraft launchers via psutil — no manual PID entry needed.

02
Memory Access

Opens the process via OpenProcess(PROCESS_VM_READ) and enumerates all committed memory regions.

03
Aho-Corasick Matching

16 MB chunks scanned in parallel threads. Thousands of signatures matched simultaneously in a single linear pass.

04
Report & Dispatch

Findings compile into an HTML report with per-client confidence scores, evidence, and a full file-journal event log.
