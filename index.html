Under the Hood

A precise, multi-stage pipeline from process discovery to detection result.

1

Process Discovery

Automatically locates all active Minecraft JVM processes on your system using system process enumeration.

2

Memory Access

Opens the process with read permissions via the Windows API, enumerating all committed, accessible memory regions.

3

Signature Matching

Each 1 MB chunk is scanned against the full signature database. Matches are flagged instantly as they're found.

4

Result Report

Clean, colour-coded output shows every detected client, its threat category, and a timestamp — ready to screenshot and share.
